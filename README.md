# Site Etec Zona Leste

Projeto escolar desenvolvido com HTML, CSS, PHP e JavaScript, inspirado no site da Etec Zona Leste.

## Tecnologias utilizadas

- HTML5
- CSS
- PHP
- JavaScript

## Páginas do site

- `index.php` → Página inicial
- `formulario.php` → Formulário com envio em PHP e validação em JavaScript
- `quem-somos.php` → Página institucional com destaques e cursos
- `contato.php` → Página com informações de contato


## Como fiz o site

Primeiro foi criada a estrutura das páginas com HTML, separando o site em Home, Formulário, Quem Somos e Contato.  
Depois foi adicionado o CSS para estilização do layout, organização dos cards, menu, imagens e responsividade.  
Em seguida foi utilizado PHP para trabalhar com as páginas `.php`, usar `echo` em diferentes partes do site e processar os dados enviados pelo formulário com o método `POST`.  
Depois foi adicionado JavaScript para deixar o site mais interativo, com carrossel de imagens, setas de navegação, rolamento, botão de pausar carrossel, botão de voltar ao topo e validação dos campos do formulário.

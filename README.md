# Site Etec Zona Leste

## Estrutura do projeto

- `index.php` → Página inicial do site
- `formulario.php` → Página de cadastro com envio via método POST
- `quem-somos.php` → Página com apresentação da escola, destaques e cursos
- `contato.php` → Página com informações de contato e imagem ilustrativa
- `style.css` → Arquivo de estilização do site
- `imagens/` → Pasta com as imagens utilizadas no projeto
